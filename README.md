# Sistema Bibliotecário

**Este projeto consiste em um sistema robusto para gestão de acervos e usuários de uma biblioteca, desenvolvido como parte da graduação em Ciência da Computação. O objetivo principal foi aplicar conceitos avançados de Programação Orientada a Objetos (POO) para criar um código escalável, organizado e de fácil manutenção.**
